# 소스에서 평가 컴포넌트 빌드

> 날짜: 2026-04-05
> 원본: `ws_aic/src/aic/docs/build_eval.md`

---

## 개요

> 대부분의 경우 **사전 빌드된 `aic_eval` Docker 컨테이너 사용을 권장**

소스 빌드가 필요한 경우: 평가 컴포넌트 수정/디버깅, 컨테이너 없는 네이티브 개발.

> 소스 변경 사항은 **공식 평가에 반영되지 않음**. 제출 컨테이너(Participant Model)만 평가됨.

## 빌드 절차

### 1. 기존 바이너리 제거 (충돌 방지)
```bash
sudo apt purge ros-kilted-ros2-control* ros-kilted-control* \
  ros-kilted-kinematics* ros-kilted-joint-state-publisher \
  ros-kilted-realtime-tools ros-kilted-gz*
```

### 2. Gazebo 저장소 추가
```bash
sudo curl https://packages.osrfoundation.org/gazebo.gpg \
  --output /usr/share/keyrings/pkgs-osrf-archive-keyring.gpg
echo "deb [...] http://packages.osrfoundation.org/gazebo/ubuntu-stable ..." | \
  sudo tee /etc/apt/sources.list.d/gazebo-stable.list
sudo apt-get update
```

### 3. 클론 및 빌드
```bash
mkdir -p ~/ws_aic/src && cd ~/ws_aic/src
git clone https://github.com/intrinsic-dev/aic
vcs import . < aic/aic.repos --recursive
rosdep install --from-paths src --ignore-src --rosdistro kilted -yr
sudo apt install -y ros-kilted-rmw-zenoh-cpp python3-pynput
source /opt/ros/kilted/setup.bash
GZ_BUILD_FROM_SOURCE=1 colcon build \
  --cmake-args -DCMAKE_BUILD_TYPE=Release \
  --merge-install --symlink-install \
  --packages-ignore lerobot_robot_aic
```

### 4. 환경변수 설정 (`~/.bashrc`에 추가)
```bash
export RMW_IMPLEMENTATION=rmw_zenoh_cpp
export ZENOH_CONFIG_OVERRIDE='transport/shared_memory/enabled=true;...'
```

---

## 실행 (3개 터미널)

```bash
# Terminal 1 — Zenoh 라우터
ros2 run rmw_zenoh_cpp rmw_zenohd

# Terminal 2 — 평가 환경
ros2 launch aic_bringup aic_gz_bringup.launch.py \
  ground_truth:=false start_aic_engine:=true

# Terminal 3 — 정책 실행
ros2 run aic_model aic_model \
  --ros-args -p use_sim_time:=true \
  -p policy:=aic_example_policies.ros.WaveArm
```
