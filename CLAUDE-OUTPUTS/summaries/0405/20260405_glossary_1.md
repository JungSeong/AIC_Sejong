# 용어집

> 날짜: 2026-04-05
> 원본: `ws_aic/src/aic/docs/glossary.md`

---

## 핵심 아키텍처

| 용어 | 설명 |
|------|------|
| **aic_engine** | Trial 전체 생애주기 관리 오케스트레이션. 태스크보드 스폰, 정책 검증, 채점 데이터 수집 |
| **aic_model** | 참가자가 구현하는 정책 노드. 센서 데이터 처리 후 로봇 명령 생성 |
| **aic_controller** | 저수준 로봇 제어 (관절/카테시안 임피던스 제어, 500 Hz) |
| **aic_adapter** | 멀티 센서 데이터를 타임스탬프 동기화해 단일 `Observation`으로 통합 |
| **Policy** | 참가자가 구현하는 알고리즘/AI 모델. 센서 → 로봇 명령 |
| **Evaluation Component** | 주최측 제공 인프라 (오케스트레이션, 컨트롤러, 센서 파이프라인 포함) |

---

## 태스크 & 환경

| 용어 | 설명 |
|------|------|
| **Cable Insertion Task** | 광섬유 케이블 커넥터를 NIC/SC 포트에 자율 삽입하는 과제 |
| **Task Board** | 4개 구역(Zone 1~4)으로 구성된 모듈식 조립 플랫폼 |
| **Zone 1** | NIC 카드 (SFP 포트) — 서버 컴퓨트 트레이 역할 |
| **Zone 2** | SC 광 포트 — 서버 랙 광 패치 패널 역할 |
| **Zone 3 & 4** | SFP 모듈, LC/SC 플러그 스테이징 픽업 위치 |
| **Trial** | 랜덤화된 설정에서 케이블 삽입 1회 실행 |

---

## 커넥터 타입

| 용어 | 설명 |
|------|------|
| **SFP** | Small Form-factor Pluggable — Zone 1 NIC 포트용 트랜시버 모듈 |
| **LC** | Lucent Connector — 소형 광섬유 커넥터 |
| **SC** | Subscriber Connector — 대형 광섬유 커넥터, Zone 2 삽입 |
| **NIC** | Network Interface Card — SFP 포트 장착 Zone 1 하드웨어 |
| **Plug** | 로봇이 파지하는 케이블 커넥터 끝단 |
| **Port** | 삽입 타겟 (SFP 포트 또는 SC 포트) |
| **Rails** | NIC / SC / 픽스처를 위치·방향 랜덤화 가능하게 고정하는 조정 레일 |

---

## 로봇 & 제어

| 용어 | 설명 |
|------|------|
| **TCP / gripper/tcp** | Tool Center Point — 그리퍼 끝단 기준 좌표 프레임 |
| **Joint-Space Control** | 관절 각도로 로봇 명령. `/aic_controller/joint_commands` |
| **Cartesian-Space Control** | 3D 포즈/속도로 로봇 명령. `/aic_controller/pose_commands` |
| **Impedance Control** | 컴플라이언스 기반 제어 — 케이블 삽입 시 힘 관리에 필수 |
| **MotionUpdate** | 카테시안 제어용 명령 메시지 |
| **JointMotionUpdate** | 관절 제어용 명령 메시지 |

---

## 감지 & 인식

| 용어 | 설명 |
|------|------|
| **Observation** | 카메라 3개 + 관절 상태 + 힘/토크 + 컨트롤러 상태 통합 스냅샷 (20 Hz) |
| **Wrist Cameras** | 손목 장착 RGB 카메라 3대 (좌/중/우) |
| **F/T Sensor** | 힘·토크 측정 센서. `/fts_broadcaster/wrench` 토픽 |

---

## 평가 지표

| 용어 | 설명 |
|------|------|
| **Jerk** | 관절 가속도의 변화율 (궤적 부드러움 지표) |
| **Convergence** | 플러그 끝단과 타겟 포트 간 유클리디안 거리 변화 |
| **Success Rate** | 케이블 완전 삽입 여부 (Tier 3 핵심 지표) |
| **Collision Penalty** | 금지구역 로봇 접촉 시 감점 |
| **Force Safety** | 과도한 삽입 힘 발생 시 감점 |

## 시뮬레이션 & 훈련

| 용어 | 설명 |
|------|------|
| **Gazebo** | 주요 평가 시뮬레이션 환경. 월드 상태 `/tmp/aic.sdf`로 내보내기 가능 |
| **IsaacLab** | NVIDIA 미러 환경 — 도메인 랜덤화 훈련용 |
| **MuJoCo** | Google DeepMind 미러 환경 — 다중 시뮬레이터 훈련용 |
| **Domain Randomization** | 여러 시뮬레이터 간 훈련으로 Sim-to-Real 전이 향상 |
| **Zenoh** | 챌린지 전용 ROS 2 미들웨어 (`rmw_zenoh_cpp`) |
