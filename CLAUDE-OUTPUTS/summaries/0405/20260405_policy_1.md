## Policy API

`Policy` 기본 클래스를 상속받아 `insert_cable()` 메서드를 구현한다.

```python
from aic_model import Policy

class MyPolicy(Policy):
    def __init__(self):
        # 모델 로딩 (PyTorch, ONNX 등)
        pass

    def insert_cable(self, task, get_observation, move_robot, send_feedback):
        while not done:
            obs = get_observation()     # 최신 Observation 획득 (최대 20 Hz)
            # obs 포함: left/center/right_image, joint_states,
            #           wrist_wrench, controller_state
            move_robot(motion_update)   # MotionUpdate 또는 JointMotionUpdate 전송
            send_feedback("status msg") # 디버깅용 피드백 (선택)
        return
```

## 새 정책 노드 튜토리얼

### 1. ROS 2 패키지 생성
```bash
pixi shell
ros2 pkg create my_policy_node --build-type ament_python
```

### 2. `package.xml` 의존성 추가
```xml
<depend>aic_control_interfaces</depend>
<depend>aic_model</depend>
<depend>aic_model_interfaces</depend>
<depend>aic_task_interfaces</depend>
```

### 3. `pixi.toml` 작성 (로컬 패키지 참조)
```toml
[package.host-dependencies]
ros-kilted-aic-model = { path = "../aic_model" }
```

### 4. 루트 `pixi.toml`에 등록
```toml
[dependencies]
ros-kilted-my-policy-node = { path = "my_policy_node" }
```

### 5. 빌드 및 실행
```bash
pixi reinstall ros-kilted-my-policy-node
pixi run ros2 run aic_model aic_model \
  --ros-args -p use_sim_time:=true -p policy:=my_policy_node.MyPolicy
```

| 타입 | 명령 |
