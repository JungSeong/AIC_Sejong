# 점수 체계

> 날짜: 2026-04-05
> 원본: `ws_aic/src/aic/docs/scoring.md`

---

## 3단계 티어 구조

| Tier | 이름 | 설명 |
|------|------|------|
| Tier 1 | 모델 유효성 | 전제 조건 — 모델 로딩 및 정상 동작 확인 |
| Tier 2 | 성능 & 수렴 | 로봇 동작 품질 정량 지표 |
| Tier 3 | 케이블 삽입 | 핵심 목표 — 삽입 성공/부분 삽입 검증 |

---

## Tier 1 — 모델 유효성 (0 or 1점)

- `InsertCable` 액션 응답 + 유효한 로봇 명령 전송 확인
- **실패 시 전체 채점 제외**

---

## Tier 2 — 성능 & 수렴

> Tier 2 긍정 점수는 **Tier 3 > 0** (삽입 성공 또는 포트 근접)일 때만 부여

| 항목 | 범위 | 기준 |
|------|------|------|
| 궤적 부드러움 | 0–6점 | Jerk 역비례. 0 m/s³→6점, ≥50 m/s³→0점 |
| 태스크 소요시간 | 0–12점 | ≤5초→12점, ≥60초→0점 |
| 궤적 효율 | 0–6점 | 경로 길이 역비례. 최소=초기 플러그-포트 거리 |
| 삽입 힘 패널티 | 0 ~ −12점 | 20 N 초과 1초 이상 지속 시 −12점 |
| 금지구역 접촉 패널티 | 0 ~ −24점 | 인클로저/태스크보드 접촉 시 −24점 |

---

## Tier 3 — 케이블 삽입 (최대 75점)

| 결과 | 점수 |
|------|------|
| 올바른 포트 완전 삽입 | **+75점** |
| 잘못된 포트 삽입 | **−12점** |
| 부분 삽입 (포트 내부, ±5mm 이내) | **38–50점** (깊이 비례) |
| 포트 근접 (포트 외부) | **0–25점** (거리 역비례) |

---

## 총점 계산

```
총점 = Tier 1 + Tier 2 + Tier 3
최대 = 1 + (6+12+6) + 75 = 100점 / Trial
```

최종 순위는 모든 Trial 점수의 누적 합산으로 결정.
