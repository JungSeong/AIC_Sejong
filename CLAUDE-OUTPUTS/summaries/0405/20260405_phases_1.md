## Qualification Phase — 모델 훈련

### 구현 방법

**권장: `aic_model` 프레임워크 사용**

1. `Policy` 클래스 상속한 Python 클래스 정의
2. `insert_cable()` 메서드 구현 (`aic_engine` 요청 시 호출)
3. 모델 로딩, `get_observation()`으로 센서 데이터 수신 (최대 20 Hz)
4. `move_robot()`으로 로봇 명령 전송
5. 태스크 완료 시 메서드 리턴

```python
class MyPolicy(Policy):
    def insert_cable(self, task, get_observation, move_robot, send_feedback):
        obs = get_observation()
        move_robot(my_command)
```

**대안: 자체 ROS 2 노드 직접 구현**
- 노드 이름: `aic_model`
- ROS 2 Lifecycle 인터페이스 구현 필수
- `/insert_cable` 액션 서버 응답 필수
- `challenge_rules.md` 요건 준수 필수

### 참가 지침

- 정책 개발 방법 제한 없음 (실물 텔레오퍼레이션, 시뮬레이터 훈련, 고전 제어 모두 가능)
- 평가: 로컬에서 직접 Gazebo 평가기 실행 가능
- 제출 후: 동일 클라우드 Gazebo 인스턴스에서 공식 채점
